FMEasyWeb: Frequently Asked Questions

What is FMEasyWeb?
FMEasyWeb is an open source solution that gives FileMaker developers the ability to quickly and easily provide browser-based access to FileMaker databases.

What functionality does EasyWeb provide?
EasyWeb provides everything you need to publish information that is stored in FileMaker databases. For example, you can use EasyWeb to publish membership and employee directories, product listings, contest results, upcoming events, and more. With EasyWeb, you can provide access to the general public or to authorized users (who will be required to login).

Is EasyWeb secure?
Yes. Security is a huge concern, and we've gone to great lengths to ensure that FMEasyWeb is as secure as possible. Click here for a summary of FMEasyWeb's security features.

Can I customize the look-and-feel of EasyWeb?
Yes. EasyWeb supports customization via CSS and easy-to-use customization files. You can control the look and feel of your solutions, change column and field names, reposition things, and more.

In addition, EasyWeb includes a few design themes that you can pick from, or you can create your own.

Do I need to know PHP in order to use EasyWeb?
No. With EasyWeb, we've done all of the hard work for you. There's no need for you to write any PHP code - or any HTML or CSS for that matter.

What changes do I need to make to my database in order to use EasyWeb?
Create a few layouts that you want EasyWeb to use and it's ready to go.

What browsers and platforms does EasyWeb support?
EasyWeb runs in any modern Web browser that supports HTML 5 and CSS, making it cross-browser and cross-platform compatible. This also means that EasyWeb will run on any modern mobile device, including those running iOS, Android, or Windows mobile.

Will I need my own Web server?
No. You can host EasyWeb on any Web server that supports PHP, which includes many of the low cost, shared Web hosting plans that Web hosting companies offer.

Is there a limit to the number of users who can connect via EasyWeb?
Based on FileMaker's technical specifications for FileMaker Server 13, up to 200 users can connect to a FileMaker database simultaneously via Custom Web Publishing (which is what EasyWeb is based on). However, in theory, up to 2,000 users can connect simultaneously.

Does EasyWeb require concurrent connection licenses?
No. EasyWeb uses Custom Web Publishing (CWP), so connections do not require or consume concurrent connections.

How much does EasyWeb cost?
FMEasyWeb is free of charge. It is being provided via an open source license.

Who is developing EasyWeb?
Tim Dietrich, Hal Gumbert, and Joe Fino.
